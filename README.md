## Swagger

When running the solution swagger will automaticaly show all possible actions:

1. POST OrdenCuenta

2. GET All OrdenCuenta

3. GET OrdenCuenta by id

4. PUT Statusfrom an existing OrdenCuenta

5. DELETE OrdenCuenta by id
